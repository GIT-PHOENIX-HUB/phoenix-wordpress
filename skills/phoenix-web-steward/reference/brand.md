# Phoenix Electric — brand & design reference

Apply this to everything created or updated on phoenixelectric.life. These values are the
live site's actual brand (extracted from the site), so matching them keeps new work
seamless with what's already there.

## Palette

| Role | Value |
|------|-------|
| Gold (primary) | `#f4cf7f` |
| Red (accent) | `#e20b00` |
| Dark / near-black | `#000000` |
| Body text | `#666666` |
| White | `#ffffff` |

- **Gold** carries primary emphasis — outlined buttons, focus states, key accents.
- **Red** is for accents that demand attention (required-field marks, alerts, active nav).
  Use it sparingly; it's a spotlight, not a background.

## Type
- **Headings and body:** `Georgia, "Times New Roman", serif`.
- Keep it readable: generous line-height, real hierarchy (don't shout with size alone).

## Buttons
- The house button is **gold-outlined**: transparent fill, `2px` gold border, dark text,
  fills gold on hover. Uppercase, slight letter-spacing. (Mirror the estimate form's
  submit button — it's the canonical example, in `plugins/phoenix-electric-estimate-form`.)

## Voice
Phoenix Electric is a **residential + commercial electrical contractor on the Colorado
Front Range.** Write like a trustworthy local electrician talking to a homeowner:

- **Plain-spoken, no hype.** No "revolutionary," no buzzwords. Clear and direct.
- **Safety- and trust-forward.** Licensed, insured, does it right.
- **Local.** Front Range / Colorado, not a faceless national.
- **Helpful over salesy.** Explain, reassure, then invite the next step (an estimate).
- Avoid heavy jargon; if a technical term is needed, say it plainly.

## Design principles (and where to go deeper)
- **Clarity and restraint** over decoration. White space is a feature.
- **One clear action per view** (e.g., "Request a free estimate").
- For deeper design direction — layout systems, hierarchy, the "Anthropic design" approach
  Shane is steering toward — consult the design material in the research corpus:
  `~/Developer/AA PROJECTS/RESEARCH PROJECTS/anthropic-docs/`. Pull from it rather than
  inventing a new look.

## When in doubt
Match what's already on the live page you're editing, and keep the gold/red/Georgia
identity intact. If a choice would change the site's *look or layout* materially, that's a
design decision — draft it and surface it to Shane, don't ship it.
